# 明日剧场 | Theatre of Tomorrow

国内首个元宇宙虚拟剧场平台，融合视讯通讯与动作捕捉技术，打造沉浸式数字艺术展演体验。

## 项目简介

「明日剧场」是一个创新的数智虚拟展演平台，将数字游戏技术、虚拟现实、人工智能与表演艺术深度融合，为用户提供沉浸式的虚拟剧场体验。平台支持虚拟展览、数字表演、互动演出等多种应用场景，推动数字艺术与科技的创新融合。

## 核心功能

- **数智虚拟展演平台** - 国内首个元宇宙虚拟剧场，融合视讯通讯与动作捕捉技术
- **AIGC数字艺术创作** - 结合人工智能技术的数字艺术创作
- **元宇宙虚拟展演** - 虚拟现实技术在艺术展演中的应用
- **沉浸式互动体验** - 支持实时互动的虚拟演出系统

## 技术栈

- **前端框架**: React 19 + Vite
- **UI组件**: Radix UI + Tailwind CSS
- **路由**: React Router DOM
- **动画**: Framer Motion
- **表单**: React Hook Form + Zod
- **图表**: Recharts
